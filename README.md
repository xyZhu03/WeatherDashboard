# Weather Dashboard



Este es un proyecto de dashboard meteorológico desarrollado en Python. La aplicación permite consultar información meteorológica y gestionar un historial de consultas.



## Estructura del Proyecto



El proyecto está organizado de forma modular para separar las responsabilidades:



- `main.py`: Punto de entrada de la aplicación.

- `weather.py`: Lógica para la obtención de datos meteorológicos.

- `historial.py`: Gestión y persistencia del historial de búsquedas.

- `menu.py`: Interfaz de usuario por consola.

- `config.py`: Configuración general del sistema.

- `repositorio.csv`: Archivo utilizado para almacenar el historial de consultas.



## Funcionalidades Principales



1. **Consulta de Clima**: Obtención de datos meteorológicos actualizados.

2. **Historial**: Registro de las búsquedas realizadas, guardado en un formato estructurado (`repositorio.csv`).

3. **Interfaz de Usuario**: Menú interactivo para navegar por las opciones del dashboard.



## Tecnologías Utilizadas



- **Lenguaje**: Python

- **Gestión de Datos**: Estructura CSV para persistencia simple.

- **Estructura**: Diseño modular para fácil mantenimiento y escalabilidad.
